Entendendo Comunicação Client x Server

Voltar

Para começar, deverá ser explicado o que são clients e servers, para que só depois seja apresentado como funciona sua comunicação.

Clients

Voltar

Os clients/clientes são os usuários que consomem informação de algum ponto. Na Web, o client é o dispositivo que está consumindo tal informação na internet.

Servers

Voltar

Os servers/servidores são super computadores sendo eles um conjunto gigante de uma empresa (conhecido como DataCenters) ou um simples computador de um quarto, que exigem mais processamento para que consiga suportar várias pessoas, vários dispositivos se comunicando com ele de uma só vez. Existem diversos tipos de servidores, exemplo:

  • Arquivos;
  • Web
  • Streaming
  • E-mail
  • segurança (Firewall)

Cada tipo de servidor centraliza o tipo de arquivo que foi direcionado para ele guardar.

Comunicação

Voltar

A comunicação se inicia quando o cliente faz requisição pela internet para um servidor, o cliente ao mandar a requisição querendo acessar um site, manda pacotes que são enviados para a internet e chegam ao servidor em que o site esta hospedado, o servidor processa os dados e depois o servidor retorna o arquivo do site para o cliente.

O que é HTML?

Voltar

O HTML ou HiperText Markup Language, traduzido para o português: Linguagem de Marcação de Hipertexto. É o componente básico da web, ele permite a inserção de conteúdo e estabelece a estrutura básica de um website. Portanto, ele serve para dar significado e organização para uma página web, para que o navegador saiba como exibir textos como elementos, carregar imagens ou outros tipos de conteúdos. Mesmo o HTML tendo "Linguagem" no seu nome, ele não é uma Linguagem de Programação e sim um arquivo de marcação.

Tags Aprendidas

Voltar

As tags são as chaves que conectam e dão significado a uma página da web, cada uma tem sua utilidade no corpo/criação de uma página web. As tags que foram apresentadas até esse ponto do bootcamp foram:

  • HTML
  • HEAD
  • BODY
  • TITLE
  • INPUT
  • I
  • STRONG
  • U
  • SUP
  • H1/H2/H3/H4...
  • P
  • A
  • BLOCKQUOTE
  • LI
  • UL
  • OL

Há várias outras oferecidas pelo bootcamp mas, irei comentar mais cobre essas.

Principais Tags

Voltar

As principais tags são aquelas que não podem faltar na criação de um Arquivo HTML pois são as chaves que as definem como prioridade.

  • HTML: Tudo que estiver dentro da tag HTML o navegador entenderá que é um arquivo HTML.
  • HEAD: São as informações que irão ser carregadas antes de exibir para o usuário.
  • BODY: É o conteúdo que será exibido para o usuário.
  • TITLE: É o nome que ficará na aba do navegador.
  • INPUT: É o que cria campos para o usuário preencher, podendo ser eles: textos, números e até cores.
  • P: Escrever parágrafos.
  • H1/H2/H3/H4...: Utilizados como titulos (h1) e subtitulos (h2 em diante).

Tags de Listagem

Voltar
  • LI: Lista Itens.
  • OL: Listagem Ordenada, lista os itens em ordem numérica.
  • UL: Listagem Não Ordenada, lista os itens em tópicos.

Tags de Modificação no Texto

Voltar
  • I: Itálico.
  • STRONG: Negrito.
  • BLOCKQUOTE: Citação.
  • U: Sublinhado.
  • SUP: Deixa o texto escrito elevado/em cima.
